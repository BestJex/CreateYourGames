# CreateYourGames！

#### 介绍
该项目是一个游戏平台，在进行游戏娱乐的基础上，还为前端开发者提供了一个提升看家本领JS的平台。普通用户可以进行游戏玩赏，榜单查看，评论游戏，喜欢游戏，分享游戏到社交软件等基本体验游戏乐趣的功能。而前端开发者用户可以查看平台中游戏的源码进行学习，并且能够在平台上发布游戏，让其他人体验自己开发的游戏。
